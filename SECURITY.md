# Security Policy

To report a security vulnerability, please follow these steps:

1. Do NOT create a public issue. Instead, email security@example.com (replace with the project's security contact) with a clear description and steps to reproduce.
2. Include the affected version, OS, and a minimal reproduction if possible.
3. The maintainers will acknowledge within 48 hours and provide a timeline for resolution.

If you are a security researcher and need a PGP key for encrypted communication, request one in your initial email.
